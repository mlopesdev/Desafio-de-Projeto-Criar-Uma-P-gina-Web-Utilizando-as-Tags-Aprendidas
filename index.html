Atributo das Tag's

Atributo da Tag é propriedade que a Tag pode adquirir, segue abaixo alguns exemplos

  • < h1 Id="titulo" > - Esta propriedade serve para identificar uma Tag
  • < h1 style="estilo" > - Esta propriedade será utilizada pelo CSS para formatar, estilizar. Neste caso o Título
  • < h1 class="titulo-principal" > - Esta propriedade será utilizada pelo CSS para formatar, todas as tag's que contenham essa classe da mesma forma

Textos

Nesta aula, aprendemos sobre algumas tag's espedificas para texto apresentadas a seguir

  • < h1 >, < h2 >, < h3 > ... < h6 > - Estas tag's são utilizadas para definir títulos ou cabeçalhos, a numeração de 1 a 6 indica a hierarquia representando por exemplo: Título, subtítulo, etc... respectivamente.
  • < p > - Esta tag serve para colocarmos o corpo do texto, separado em paragrafos
  • < blockquote > - Esta tag serve para colocarmos um texto como uma citação
    ESTE TEXTO É UMA CITAÇÃO
  • < MARK > - Esta tag serve marcarmos um texto como se estivessemos usado uma caneta marca texto
  • < u > - Esta tag serve para deixar um texto sublinhado

Listas ordenadas e não ordenadas

As tag's de lista ordenadas tem a seguinte formatação:

< ol > - Esta tag define a lista ordenada, sendo necessário uma tag filha para definir o que vai ser exibido dentro da lista
< li > - Tag filha, o que estiver entre destas tag's serão considerados item da lista e automaticamente serão enumeradas.
< /li >
< /ol >

Ficando com a seguinte aparência:

  1. Maçã
  2. Pera
  3. Mamão

As tag's de listas não ordenadas tem a mesma formatação acima, apenas alterando a tag PAI por < ul >

Link's

Nestas aulas foram apresentadas 3 tipos de link's que tem a seguinte formatação com suas respectivas finalidades

< a href="http://dio.me">dio.me< /a >   Gerando um link como esse: dio.me que direciona para o site da dio nesta mesma aba.

< a href="http://dio.me" target="_blank">dio.me< /a >   Gerando um link como esse: dio.me que direciona para o site da dio em uma nova aba.

< a href="#inicio"> (voltar) < /a >   Gerando um link como esse: (voltar) que direciona para uma parte específica desta mesma pagina, neste caso, o título principal desta página está com ao atributo id="inicio"
